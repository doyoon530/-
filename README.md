# KMEANS
## 설명
IMU센서를 활용한 자이로값 기반 양치 구역 추정 클러스터링 코드입니다.

sampling.ipynb :  동일한 개수로 각 구역의 데이터를 샘플링하는 코드입니다.

clustering.ipynb :  클러스터링을 진행하는 코드입니다.
